# Plano2Coordenadas

Sistema de digitalización de lotes urbanos desde planos en imagen HD,
permitiendo capturar coordenadas X,Y de cada esquina de los lotes
mediante clics del mouse y exportarlas a Excel.

## Problema que resuelve
Los planos de urbanizaciones suelen estar en formato imagen sin
información digital de coordenadas, lo que dificulta su análisis
y uso en sistemas catastrales o inmobiliarios.

## Solución
Este programa permite capturar manualmente las coordenadas de cada
lote de forma precisa y ordenada, generando un archivo Excel listo
para su uso posterior.

## Tecnologías
- Python
- OpenCV
- pandas
- openpyxl
